# Advanced Prompt Patterns

Использовать только при высокой сложности задачи.

---

## 1️⃣ Multi-Expert Pattern

### Когда использовать
- Стратегия
- Архитектура
- Сложная аналитика

### Механизм
1. Сгенерировать до 3 экспертов.
2. Каждый формулирует:
   - ключевые факторы
   - риски
   - рекомендации
3. Выполнить синтез.
4. Убрать внутреннюю дискуссию из финального ответа.

---

## 2️⃣ Meta-Prompt Pattern

### Когда использовать
- Оптимизация промптов
- Создание промптов для других моделей

### Структура
```
Проанализируй следующий промпт:
[текст]

Улучши его по:
- ясности
- структуре
- токен-эффективности
```

---

## 3️⃣ Self-Refinement Pattern

### Когда использовать
- Требуется высокая точность
- Модель склонна к ошибкам

### Структура
1. Сгенерировать ответ.
2. Проверить на соответствие цели.
3. Исправить слабые места.
4. Вывести финальную версию.

---

## 4️⃣ Decomposition Pattern

### Когда использовать
- Большая задача
- Много этапов

### Структура
```
Разбей задачу на этапы.
Реши каждый отдельно.
Собери итоговый результат.
```

---

## 5️⃣ Evaluation Pattern

### Когда использовать
- Проверка качества
- A/B тестирование

### Структура
```
Оцени ответ по:
- полноте
- точности
- структуре
- избыточности

Дай рекомендации по улучшению.
```

---

# Правило использования Advanced

- Не применять к простым задачам.
- Максимум 1–2 продвинутых паттерна одновременно.
- Всегда завершать финальным синтезом.
